Four directions for an elevated sprintassembly.com

Every variant below carries the current site’s copy verbatim — same words, same sections, same hero, same team bios, same LinkedIn insights. What changes is how the information is presented: typography, layout, rhythm, density, and signature visual moves.

All four stay true to the brand brief — rigorous, pragmatic, unfussy — and all four use the warm near-white surface with mossy green #afb975 as the single brand accent. None use banned reflex fonts (Inter, IBM Plex, Fraunces, Instrument, DM Sans, Space Grotesk). None go dark mode. None introduce consultancy grey or AI-startup futurism.

iv. Minimal Document Open →

The site as an investor memo.

Maximum restraint. Small body type, generous leading, enormous margins between sections. Type does all the work — no cards, no shadows, no chrome. Moss green appears once or twice per section as a thin rule or a small dot. The most grown-up of the four.

Round 2 · V2 variations · sectional separation + color exploration

Four variations on the refined V2.

All four start from the V2 you signed off on. Each takes a different swing at the same two questions: how do we visually separate sections on a single warm background, and how does the color palette evolve beyond strict black/moss/beige.

a. Tinted Sections Open →

Separation by paper tint.

Most restrained. Each section gets its own subtle paper colour — amber for Opportunity, cool teal for Process, warm cream for Team, muted sage for Insights, deeper amber for Contact. Same brand palette, just spread across zones. The colour shift itself is the section break.

Sep’nSection background tint
ColorBlack / moss / beige (zoned variants)
LoudnessQuiet · the most disciplined
b. Spot Multi-color Open →

One colour per section.

Each section claims its own accent colour. Moss for Opportunity, terracotta for Process, dusty blue for Team, gold for Insights, ink-black for Contact. The slab in each section title plus a short 72px offset rule at the section’s top inherits that colour. Multi-colour without becoming a rainbow.

Sep’n72px offset rule + colour shift
ColorMoss · terracotta · blue · gold · ink
LoudnessMedium · expressive but cohesive
c. Brutalist Open →

Inverse black acts.

The loudest. Process and Contact go full-black inverse — paper text, moss accents popping harder against the dark. Hero and section titles get bigger. The page reads as alternating acts: paper, paper, BLACK, paper, paper, BLACK. Strong visual drumbeat.

Sep’nFull inverse sections (Process & Contact)
ColorBlack ↔ paper, moss spot
LoudnessHigh · aggressive · committed
d. Masked Type Open →

Image-driven accent slabs.

Most experimental. Hero’s “Agentic Scale” gets a conic rainbow representing AI’s breadth. Each section title accent uses a different image-driven mask: striped scan-pattern on Process, dot halftone on Team, crosshatch grid on Insights, deep ink radial on Contact. A short 56px hairline marks the start of each section. Type as image.

How to read these

The top four are the original wide-exploration directions (Editorial, Index & Ledger, Oversize, Minimal). The bottom four all start from the refined V2 you preferred — same structure, same content, same fonts. Only section separation and colour treatment change.

Pay attention to how each round-2 variant handles the transition between Opportunity and Process — that’s where separation choices show up most clearly. Then tell me which direction (or hybrid) is closest. I can keep iterating, or commit to one and start porting it into the React codebase.
